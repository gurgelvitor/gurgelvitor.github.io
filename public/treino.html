Ajuda - Montador de Treinos

Como Começar

1
Clique em "Adicionar Dia de Treino" para criar um novo dia
2
Dê um nome para o dia (ex: "Segunda - Peito")
3
Adicione exercícios clicando em "Adicionar Exercício"
4
Configure séries, repetições e cargas para cada exercício

Adicionando Exercícios

1
Use o campo de busca para encontrar exercícios
2
Selecione o exercício desejado na lista
3
Defina séries, repetições, carga e intervalo
4
Clique em "Adicionar" para incluir no treino
💡 Dica Profissional
Organize os exercícios por músculos trabalhados para melhor fluxo do treino

Estrutura do Treino

Dias de Treino: Organize por grupos musculares ou objetivos

Ordem dos Exercícios: Comece com exercícios compostos

Progressão: Aumente gradualmente a carga e volume

1
Exercícios grandes (agachamento, supino)
2
Exercícios isolados (rosca, elevação lateral)
3
Exercícios de estabilização (prancha, abdominal)

Atribuição a Alunos

1
Salve seu treino para acesso futuro
2
Selecione o aluno na lista
3
Dê um nome para a ficha de treino
4
O treino ficará disponível para o aluno

Anamneses

Crie questionários personalizados para avaliar seus alunos:

1
Clique em "Nova Anamnese" para criar um questionário
2
Defina um título e descrição para a anamnese
3
Adicione perguntas personalizadas (saúde, objetivos, histórico)
4
Salve e atribua aos alunos para responderem
📋 Tipos de Perguntas
Use perguntas abertas para detalhes e fechadas para dados específicos (idade, peso, altura)
🎯 Importância
As anamneses ajudam a personalizar treinos e dietas conforme as necessidades individuais

Funções Adicionais

Imprimir Ficha: Gera versão para impressão com todos os detalhes

Salvar Treino: Armazena para uso futuro

Duplicar Dia: Cria cópia de um dia existente

Relatório de Cargas: Acompanhe evolução na página de Cargas

🎯 Organização
Use nomes descritivos para facilitar a identificação posterior
📊 Acompanhamento
Após criar treinos, use a página Cargas para registrar evolução e gerar relatórios PDF

Dicas de Montagem

⚖️ Equilíbrio Muscular
Trabalhe grupos musculares antagonistas para evitar desequilíbrios
🔄 Variedade
Altere exercícios periodicamente para evitar adaptações
⏱️ Descanso
Respeite os intervalos para melhor recuperação e performance
